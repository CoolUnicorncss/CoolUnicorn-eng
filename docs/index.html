Quick Start

Copy-paste the stylesheet 
<link>
into your 
<head>
before all other stylesheets to load our CSS. 
<link href="https://coolunicorn.ml/style.css" rel="stylesheet" type="text/css" />

Or download this file!


Starter Template

Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this: 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width">
    <title>CoolUnicorn.css</title>
    <link href="https://coolunicorn.ml/style.css" rel="stylesheet" type="text/css" />
  </head>
  <body>
    <nav class="nav">
      <div class="nav-item">
        <img src="https://coolunicorn.ml/icon.png" class="nav-img">
      </div>
      <div class="nav-item">
        <a class="nav-link" href>CoolUnicorn.css</a>
      </div>
      <div class="nav-item">
        <button class="nav-button"  onClick="location.replace('/docs/')">get started</button>
      </div>
    </nav>
    <div class="container floating">
      <h1 class="title">CoolUnicorn.css</h1>
      <p class="subtitle">slow, un-responsive and<br> most importantly, looks good.</p>
      <button class="big-button" onClick="location.replace('/docs/')">get started</button>
    </div>
    <script src="script.js"></script>
  </body>
</html>
